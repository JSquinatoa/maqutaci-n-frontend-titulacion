Diseña una aplicación web SaaS moderna para un sistema de gestión odontológica llamado “Odonto Estetic Cloud”.

La interfaz debe tener un diseño premium, minimalista, moderno y corporativo, inspirado visualmente en:
https://prium.github.io/phoenix/v1.24.0/index.html

IMPORTANTE:

* NO copiar el diseño exacto.
* Solo usarlo como referencia visual de estructura, spacing, sidebar, cards, dashboard y estilo general.
* El resultado debe verse más moderno, limpio y profesional.
* Diseño responsive desktop-first.
* UI/UX enfocada en clínicas odontológicas privadas.
